일반 대화 vs 맞춤형 Gem

제미나이의 Gem은 반복해서 사용할 기준·자료·작업 방식을 미리 정해 놓는 나만의 작업 공간입니다.

일반 대화맞춤형 Gem
매번 조건을 다시 설명공통 기준을 지침으로 저장
여러 작업이 섞일 수 있음주제별 작업을 한곳에 관리
이전 자료를 다시 찾아야 함참고 자료를 모아 활용
일회성 작업에 적합반복 업무에 적합
프롬프트는 한 번의 작업을 위한 요청,
Gem 지침은 반복될 작업의 공통 원칙입니다.

2단계 — 지침 생성기 질문에 답하면 지침이 완성됩니다

이 슬라이드 자료에 들어 있는 ✨ 나만의 브리핑 Gem 지침 생성기에서 질문 5개에 답하면, 복사해서 바로 쓸 수 있는 지침이 만들어집니다.

단계지침 생성기가 묻는 것
1단계직업·분야 — 나는 어떤 일을 하는 사람인가
2단계챙겨보는 정보 주제 — 어떤 소식을 놓치면 안 되는가
3단계정보를 쓰는 목적 — 수업 준비 · 경영 판단 · 고객 응대 · 면접 준비 · 업무 보고 · 개인 공부
4단계선별 기준 — 우선해서 볼 정보 / 걸러낼 정보
5단계브리핑 확인 상황 — 아침 스마트폰 · 회의 직전 · 고객/팀 공유
생성답변을 모아 복사해 붙여 넣을 수 있는 완성 지침 생성
작성 원칙과 브리핑 템플릿은 고정되어 자동으로 포함됩니다 — 여러분은 분야·목적·기준만 정하면 됩니다.

  (지정이 없으면 가장 중요한 소식)을 골라

3차 — 이미지 제작 기능 붙이기

같은 Gem에 다음 슬라이드의 추가 지침을 복사 → 지침 맨 아래에 붙여넣기 → 저장하면, 브리핑에서 이미지까지 한 번에 됩니다. 명령어 4개가 추가됩니다.

명령어하는 일
포스터공지·홍보·블로그 대문용 대표 이미지 1장 즉시 제작 (가로형 16:9, 제목은 짧고 크게)

왜 한 장씩? — 6장을 한 번에 요청하면 한 이미지에 합쳐지거나, 일부만 나오거나, 디자인이 달라질 수 있습니다. 내용은 한 번에 기획, 이미지는 한 페이지씩이 가장 안정적입니다.

이미지 제작 추가 지침 — 복사해서 내 Gem 지침 아래에 붙여넣기

[이미지 제작 기능 추가]

아래 명령어가 입력되면 방금 브리핑·심층 분석한 내용
(또는 사용자가 붙여넣은 글)을 시각 콘텐츠로 만든다.

이미지를 만들기 전에 먼저 다음을 파악한다.
- 글의 주제
- 주요 독자
- 핵심 메시지
- 이미지에 반드시 필요한 문구

자료에 없는 날짜, 수치, 서비스, 인물, 혜택은 만들지 않는다.

[명령어 1: 포스터]
사용자가 '포스터'라고 입력하면
공지·홍보·블로그 대문에 쓸 대표 이미지를 즉시 제작한다.
- 비율: 가로형 16:9
- 용도: 대표 이미지(단톡방 공지·SNS·블로그 대문)
- 제목은 짧고 크게 표시
- 한글 문구는 제목과 짧은 보조 문구만 사용
- 본문 전체를 이미지에 넣지 않는다.
- 스마트폰에서도 제목이 잘 보이게 한다.
- 글의 분위기와 주요 독자에게 맞는 색상을 사용한다.
- 이미지 생성 후 들어간 한글 문구의 오류를 점검한다.
- 설명이나 이미지 프롬프트만 제시하지 말고 실제 이미지를 생성한다.

[명령어 2: 카드뉴스]
사용자가 '카드뉴스'라고 입력하면
입력한 글을 총 6페이지 카드뉴스로 기획한다.
- 페이지 비율: 세로형 4:5
- 한 페이지에는 하나의 핵심 메시지만 사용
- 모든 페이지의 색상, 글꼴, 여백, 삽화 스타일을 통일
- 긴 문장은 짧은 카드뉴스 문장으로 바꾼다.
- 원문에 없는 사실과 수치를 추가하지 않는다.

페이지 구성:
1페이지: 독자의 관심을 끄는 표지
2페이지: 독자가 겪고 있는 문제
3페이지: 핵심 개념 또는 원인
4페이지: 실제 사례나 좋은 예와 나쁜 예
5페이지: 따라 할 수 있는 방법 또는 체크리스트
6페이지: 핵심 요약과 행동 제안

먼저 다음 표를 제시한다.
- 페이지 번호 / 페이지 제목 / 핵심 문구 / 보조 문구 / 추천 이미지 / 강조할 단어

구성안이 완성되면
'카드뉴스 제작 시작이라고 입력하면 1페이지부터 제작합니다.'
라고 안내한다.

[명령어 3: 카드뉴스 제작 시작]
1페이지 이미지만 생성한다.
한 화면에 여러 페이지를 합치지 않는다.
이미지 생성 후 페이지 번호와 한글 오류를 확인한다.

[명령어 4: 계속]
앞에서 확정한 디자인을 유지하여
다음 페이지 한 장만 생성한다.
6페이지까지 순서대로 진행한다.

[이미지 제작 원칙]
- 각 페이지는 독립된 이미지 한 장으로 제작한다.
- 6페이지를 한 이미지에 배치하지 않는다.
- 한글 문구를 최소화하여 글자 오류를 줄인다.
- 이미지마다 인물과 디자인 분위기가 갑자기 바뀌지 않게 한다.
- 생성 결과에 글자 오류가 있으면 해당 페이지만 수정한다.
- 특정 업체의 로고나 저작권 이미지를 임의로 넣지 않는다.
- 사용자가 제공한 인물 사진은 동의된 용도로만 사용한다.
복사 → 내 Gem 편집(연필 아이콘) → 지침 맨 아래에 붙여넣기 → 저장. 새 Gem을 만들 필요가 없습니다!
